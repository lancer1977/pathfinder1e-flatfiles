# Pathfinder1e Flatfiles

This directory holds generated Pathfinder 1e JSON flatfiles.

## Coverage

- Every root `~/code/DataSeeds/pathfinder1e/*.js` payload is mirrored here as a `.json` file.
- The mirrored shape intentionally stays compatible with the existing payloads.
- Families can gain additional fields when a refresh script enriches them from the PF1 markdown vault.
- Families can gain `Srd*` overlay fields when matching records are found in the upstream SRD markdown repository.
- Duplicate-name records without upstream IDs can gain `FlatfileId` values on second and later occurrences so database imports keep all records.

## Enriched Families

- `characterClass.json`
- `skills.json`
- `spells.json`

These files are regenerated by family-specific scripts after the full DataSeeds sync.

## SRD Markdown Overlay

- `spells.json`
- `monster.json`

Run `npm run update:srd` to pull the latest upstream markdown cache and update matching records.
The overlay preserves existing DataSeeds fields and adds `Srd*` fields for source path, title, source metadata, structured spell values, statblock values, and descriptions.
